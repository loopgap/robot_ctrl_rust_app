This page is the local HTML help entry used by robot_control_rust and rust_tools_suite.

默认界面缩放基线为 150%。推荐使用 Ctrl + 鼠标滚轮 快速缩放；设置中的滑块只调整待应用值，点击“应用缩放”后才真正生效。

应用组成

  • robot_control_rust：串口/协议/控制算法主界面。
  • rust_tools_suite：CSV、JWT、Regex、JSON、校验和、URL/Base64/UUID 等工具合集。
  • rust_indie_tools：已退役，不再作为正式工作区或发布对象保留。

缩放与字体

  • 默认缩放：150%
  • 可调范围：100% 到 220%
  • 快捷操作：Ctrl + 滚轮
  • 设置面板中的滑块不会在拖动时实时抖动界面。

robot_control_rust

  • 用于串口通信、协议分析、报文构造、PID/NN 调参与数据可视化。
  • 帮助菜单中的“文档”会优先打开本地 HTML 帮助页。
  • 更新检查如果没有独立说明页，也会回退到本地帮助页，而不是打开无意义的 GitHub 搜索页。

rust_tools_suite

  • 统一承接原先零散的小工具能力。
  • 适合做文本处理、编码转换、CSV 清洗、JWT 解析和 Regex 验证。
  • 帮助菜单中的“文档”同样优先打开这份本地 HTML 页面。

工作流与发布

  • 本地自动化入口：.\make.ps1 workflow-seal、.\make.ps1 check、.\make.ps1 preflight
  • 发布资产应包含：robot_control_rust.exe、rust_tools_suite.exe、RobotControlSuite_Setup.exe、checksums-sha256.txt
  • 安装包会同时放入两个可执行文件，并保留本地 HTML 帮助页。

排查建议

  • 若帮助页没有打开，优先检查安装目录或可执行文件同级是否存在 help_index.html。
  • 若字体仍偏小，先确认当前缩放是否已应用，而不是仅修改了待应用值。
  • 若需要重新验证工作区状态，先执行 .\make.ps1 workspace-guard。

# Regex 巡检工坊

> 用于日志筛选、文本巡检和规则回归

## 功能概述

| 功能 | 说明 | 场景 |
|------|------|------|
| 正则输入 | 支持 Rust Regex 语法 | 告警规则快速验证 |
| 多行文本匹配 | 面向日志和批量文本 | 故障排查 |
| 命中统计 | 输出命中行数量 | 规则效果评估 |
| 命中结果复制 | 一键复制匹配输出 | 下游分析或归档 |

## 使用流程

1. 输入正则表达式。
2. 粘贴待匹配文本。
3. 执行匹配并检查命中统计。
4. 复制命中结果到下游系统。

## 闭环状态

输入 -> 校验 -> 执行 -> 验证 -> 导出

# JWT 解析工坊

> 快速查看 JWT Header/Payload（不执行签名验签）

## 功能概述

| 功能 | 说明 | 场景 |
|------|------|------|
| Header 解析 | Base64URL 解码并格式化 | 头部算法字段检查 |
| Payload 解析 | Base64URL 解码并格式化 | Claims 内容核验 |
| 错误提示 | 对格式和编码错误给出反馈 | 联调阶段快速定位 |
| 结果复制 | 一键复制 Header + Payload | 研发协作与问题记录 |

## 使用流程

1. 粘贴 JWT 字符串。
2. 点击解析。
3. 查看 Header 与 Payload。
4. 复制结果用于联调记录。

## 注意事项

- 当前工具仅做结构解析，不做签名合法性校验。
- 若 token 分段或编码异常，会在状态区给出错误提示。

# CSV 清洗工坊

> 面向导入前数据预处理的轻量清洗工具

## 功能概述

| 功能 | 说明 | 场景 |
|------|------|------|
| 列宽一致性检查 | 识别列数不一致的行 | 批量导入前校验 |
| 可选去重 | 按整行内容去重 | 去除重复记录 |
| 清洗输出复制 | 一键复制结果 | 快速粘贴到下游系统 |

## 使用流程

1. 粘贴原始 CSV 文本。
2. 按需勾选去重。
3. 点击执行清洗，查看一致性结果。
4. 复制输出并导入下游系统。

## 闭环状态

输入 -> 校验 -> 执行 -> 验证 -> 导出

# 微型工具集 - 概述

> 面向实际交付场景的 Rust 小工具合集

## 项目简介

`rust_micro_tools` 是一个统一的 GUI 工具入口，包含 **10 款高频工具**，全部支持双语界面和偏好持久化。

## 工具清单

| 工具 | 功能 | 场景 |
|------|------|------|
| 校验和工坊 | CRC32/FNV1a-64/SHA256 | 接口联调、报文验签 |
| JSON 工坊 | JSON 校验/格式化/压缩 | 配置治理、跨环境分发 |
| 日志巡检 | 正则筛选、命中统计 | 故障定位、告警复核 |
| URL 编解码 | URL Encode/Decode | 参数透传、链接联调 |
| 时间戳转换 | Unix 时间戳互转 | 日志对时、链路回放 |
| Base64 工坊 | Base64 编解码 | Token 排障、二进制联调 |
| UUID 批量生成 | 批量 UUID 生成 | 测试数据、主键生成 |
| CSV 清洗工坊 | 列宽一致性检查、去重、输出复制 | 数据导入前预处理 |
| JWT 解析工坊 | Header/Payload 解析（不验签） | Token 联调与字段核验 |
| Regex 巡检工坊 | 正则匹配、命中统计、输出复制 | 日志筛选、规则回归 |

## 技术特性

- **双语界面**: 中文 / English 实时切换
- **语言持久化**: 重启后自动恢复
- **闭环流程**: 输入 → 校验 → 执行 → 验证 → 导出
- **统一主题**: 深色工业风 + 蓝色强调色

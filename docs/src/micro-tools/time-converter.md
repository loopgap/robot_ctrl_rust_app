# 时间戳转换

> Unix 时间戳与时间字符串互转工具

## 功能概述

| 功能 | 说明 |
|------|------|
| Unix 秒 → 时间字符串 | 10 位时间戳转可读格式 |
| Unix 毫秒 → 时间字符串 | 13 位时间戳转可读格式 |
| 时间字符串 → Unix 秒 | 字符串转 10 位时间戳 |
| 多时区支持 | UTC、本地时区、自定义 |

## 时间格式

### 支持的输入格式

| 格式 | 示例 |
|------|------|
| Unix 秒 | `1705312345` |
| Unix 毫秒 | `1705312345123` |
| ISO 8601 | `2024-01-15T10:23:45Z` |
| 日期时间 | `2024-01-15 10:23:45` |
| 日期 | `2024/01/15 10:23:45` |

### 输出格式

| 格式 | 示例 |
|------|------|
| ISO 8601 | `2024-01-15T10:23:45+08:00` |
| 日期时间 | `2024-01-15 10:23:45` |
| UTC | `2024-01-15 02:23:45` |

## 使用方法

### 交互模式

```powershell
cargo run -- time
```

### 命令行模式

```powershell
# Unix 秒转时间
cargo run --release -- time --from-unix 1705312345

# Unix 毫秒转时间
cargo run --release -- time --from-unix-ms 1705312345123

# 时间转 Unix 秒
cargo run --release -- time --to-unix "2024-01-15 10:23:45"

# 指定时区
cargo run --release -- time --from-unix 1705312345 --timezone "Asia/Shanghai"
```

### 参数选项

| 参数 | 说明 |
|------|------|
| `--from-unix` | 从 Unix 秒转换 |
| `--from-unix-ms` | 从 Unix 毫秒转换 |
| `--to-unix` | 转换为 Unix 秒 |
| `--timezone` | 指定时区 |
| `--format` | 输出格式 |

## 典型应用场景

### 场景 1: 日志时间对时

```powershell
# 日志时间
2024-01-15 10:23:45.123 INFO Server started

# 转换为 Unix 时间戳
cargo run --release -- time --to-unix "2024-01-15 10:23:45"
# 输出: 1705312425
```

### 场景 2: 链路回放

```powershell
# 获取时间戳范围
cargo run --release -- time --from-unix 1705312345
cargo run --release -- time --from-unix 1705315945
# 差值 = 1 小时
```

### 场景 3: 跨时区问题排查

```powershell
# 欧洲团队日志 (UTC+1)
2024-01-15T11:23:45+01:00

# 转换为北京时间
cargo run --release -- time --from-unix 1705312425 --timezone "Asia/Shanghai"
# 输出: 2024-01-15 18:23:45
```

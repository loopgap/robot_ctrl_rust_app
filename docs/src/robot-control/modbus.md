# Modbus Tools - Modbus 工具

> Modbus RTU / Modbus TCP 双协议支持

## 功能概述

Modbus 工具提供 Modbus RTU 和 Modbus TCP 两种协议的支持。

### 支持的协议

| 协议 | 接口 | 说明 |
|------|------|------|
| Modbus RTU | RS485 / RS232 | 串行接口 |
| Modbus TCP | Ethernet | TCP/IP 接口 |

## 寄存器类型

### 四种寄存器类型

| 类型 | 功能码 | 访问 | 说明 |
|------|--------|------|------|
| 线圈 (Coil) | 01, 05, 15 | 读写 | 单个位 |
| 离散输入 (Discrete Input) | 02 | 只读 | 单个位 |
| 保持寄存器 (Holding Register) | 03, 06, 16 | 读写 | 16-bit 字 |
| 输入寄存器 (Input Register) | 04 | 只读 | 16-bit 字 |

## 功能码

### 读写操作

| 功能码 | 名称 | 操作 | 数据大小 |
|--------|------|------|----------|
| 01 | Read Coils | 读多个线圈 | 位 |
| 02 | Read Discrete Inputs | 读离散输入 | 位 |
| 03 | Read Holding Registers | 读保持寄存器 | 16-bit 字 |
| 04 | Read Input Registers | 读输入寄存器 | 16-bit 字 |
| 05 | Write Single Coil | 写单个线圈 | 位 |
| 06 | Write Single Register | 写单个寄存器 | 16-bit 字 |
| 15 | Write Multiple Coils | 写多个线圈 | 位 |
| 16 | Write Multiple Registers | 写多个寄存器 | 16-bit 字 |

| 操作 | 说明 |
|------|------|
| 批量读取 | 一次读取多个寄存器 |
| 批量写入 | 一次写入多个寄存器 |
| 扫描 | 自动扫描从机地址 |

## 配置参数

### RTU 配置

| 参数 | 说明 | 常用值 |
|------|------|--------|
| 从机地址 | 目标设备地址 | 1-247 |
| 波特率 | 通信速率 | 9600, 19200, 38400 |
| 数据位 | 数据位数 | 8 |
| 停止位 | 停止位数 | 1 或 2 |
| 校验位 | 校验方式 | None, Odd, Even |

### TCP 配置

| 参数 | 说明 |
|------|------|
| 从机地址 | 目标 IP 地址 |
| 端口号 | Modbus 端口 (默认 502) |
| 从机 ID | Modbus 单元标识符 |

## 使用示例

### 读取保持寄存器

```
功能码: 03
起始地址: 0x0000
寄存器数量: 10

请求: 01 03 00 00 00 0A C5 CD
响应: 01 03 14 00 01 00 02 00 03 00 04 00 05 00 06 00 07 00 08 00 09 00 0A XX XX
```

### 写入单个寄存器

```
功能码: 06
寄存器地址: 0x0001
写入值: 0x1234

请求: 01 06 00 01 12 34 99 BE
响应: 01 06 00 01 12 34 99 BE
```

## 数据类型转换

### 16-bit 寄存器组合

| 数据类型 | 需要的寄存器数 | 说明 |
|----------|---------------|------|
| u16 | 1 | 无符号 16 位整数 |
| i16 | 1 | 有符号 16 位整数 |
| u32 | 2 | 无符号 32 位整数 |
| i32 | 2 | 有符号 32 位整数 |
| f32 | 2 | 单精度浮点数 |
| u64 | 4 | 无符号 64 位整数 |
| f64 | 4 | 双精度浮点数 |
| string | N | 字符串 (每个寄存器 2 字节) |

### 字节序

| 字节序 | 说明 | 寄存器顺序 |
|--------|------|------------|
| ABCD | 大端 | 高字在高地址 |
| BADC | Big-endian swap | 高字在低地址 |
| CDAB | Little-endian swap | 低字在高地址 |
| DCBA | 小端 | 低字在低地址 |

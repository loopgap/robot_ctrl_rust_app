# Data Visualization - 数据可视化

> 实时波形、统计图表与通道管理

## 功能概述

数据可视化模块提供多通道实时数据可视化能力，支持 6 种可视化类型。

### 可视化类型

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| 折线图 | 实时波形显示 | 趋势监控 |
| 散点图 | 数据点分布 | 分布分析 |
| 柱状图 | 分类对比 | 静态对比 |
| 仪表盘 | 数值仪表 | 单值监控 |
| 直方图 | 频率分布 | 统计分析 |
| 表格 | 数据列表 | 详细数据 |

## 通道管理

### 数据源类型

| 数据源 | 说明 |
|--------|------|
| RobotState | 机器人状态字段 (位置、速度、电流等) |
| PacketField | 报文模板的特定字段 |
| RawOffset | 原始数据的指定偏移量 |

### RobotState 字段

| 字段 | 说明 | 单位 |
|------|------|------|
| position | 位置 | mm |
| velocity | 速度 | mm/s |
| acceleration | 加速度 | mm/s² |
| current | 电流 | A |
| voltage | 电压 | V |
| temperature | 温度 | °C |

### 通道配置

| 参数 | 说明 |
|------|------|
| 名称 | 通道显示名称 |
| 数据源 | 数据来源类型和路径 |
| 可视化类型 | 图表类型 |
| 颜色 | 图表线条颜色 |
| 启用/禁用 | 是否显示 |

## 实时图表

### 折线图配置

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 窗口大小 | 显示的数据点数 | 500 |
| 刷新率 | 图表更新频率 | 60 Hz |
| Y 轴范围 | 自动/手动设置 | 自动 |
| X 轴范围 | 时间窗口 | 10 秒 |

### 多通道叠加

支持在同一图表上叠加显示多个通道：

```
┌──────────────────────────────────────────────┐
│  Channel 1 (position)  ─── 蓝色             │
│  Channel 2 (velocity)  ─── 红色             │
│  Channel 3 (current)   ─── 绿色             │
└──────────────────────────────────────────────┘
```

## 统计分析

### 统计指标

| 指标 | 说明 |
|------|------|
| 当前值 | 最新数据点 |
| 最大值 | 统计周期内最大值 |
| 最小值 | 统计周期内最小值 |
| 平均值 | 统计周期内平均值 |
| 标准差 | 数据离散程度 |

### 统计计算

```rust
let current = buffer.last().unwrap_or(&0.0);
let max = buffer.iter().cloned().fold(f64::MIN, f64::max);
let min = buffer.iter().cloned().fold(f64::MAX, f64::min);
let avg = buffer.iter().sum::<f64>() / buffer.len() as f64;
```

## 使用示例

### 添加新通道

1. 点击 "添加通道"
2. 选择数据源类型
3. 选择具体字段或配置偏移
4. 选择可视化类型
5. 设置颜色和名称
6. 确认添加

### 修改通道配置

1. 在通道列表中选择要修改的通道
2. 修改可视化类型
3. 修改颜色
4. 修改名称
5. 更改会自动保存

### 删除通道

1. 点击通道旁边的删除按钮
2. 确认删除
3. 通道及数据缓冲区将被移除

## 导出功能

### 数据导出

支持导出通道历史数据：

| 格式 | 说明 |
|------|------|
| CSV | 逗号分隔值 |
| JSON | 结构化 JSON |
| TXT | 纯文本 |

### 导出选项

| 选项 | 说明 |
|------|------|
| 时间范围 | 选择导出时间段 |
| 通道选择 | 选择要导出的通道 |
| 时间戳 | 是否包含时间戳 |
| 下采样 | 降低导出数据密度 |

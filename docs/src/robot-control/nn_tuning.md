# NN Tuning - 神经网络调参

> 神经网络辅助参数整定与 LLM 建议

## 功能概述

NN Tuning 模块提供基于神经网络的参数自动整定能力，并与 LLM API 集成获取调参建议。

### 主要功能

| 功能 | 说明 |
|------|------|
| 神经网络调参 | 使用 NN 学习最优控制参数 |
| 梯度监控 | 实时显示梯度变化 |
| LLM 建议 | 获取 AI 辅助调参建议 |
| 参数预览 | 查看推荐参数值 |

## 神经网络配置

### 网络结构

| 参数 | 说明 |
|------|------|
| 输入层 | 误差、误差变化率、当前参数 |
| 隐藏层 | 可配置神经元数量 |
| 输出层 | 调整后的 Kp, Ki, Kd |

### 训练参数

| 参数 | 说明 | 典型值 |
|------|------|--------|
| 学习率 | 参数更新步长 | 0.001-0.1 |
| 动量 | 加速收敛 | 0.9 |
| 批量大小 | 每次训练样本数 | 32 |
| 训练周期 | 迭代次数 | 100-1000 |

## 调参流程

### 自动调参步骤

1. **数据采集**: 收集系统响应数据
2. **模型训练**: 使用 NN 学习最优参数
3. **参数验证**: 在系统中测试推荐参数
4. **微调优化**: 根据实际效果调整

### 调参指标

| 指标 | 说明 | 目标 |
|------|------|------|
| IAE | 积分绝对误差 | 越小越好 |
| ISE | 积分平方误差 | 越小越好 |
| ITAE | 积分时间加权绝对误差 | 越小越好 |
| 超调量 | 最大偏差百分比 | < 5% |
| 调节时间 | 达到稳态时间 | 越短越好 |

## LLM 集成

### API 配置

| 参数 | 说明 |
|------|------|
| API URL | LLM 服务端点 |
| API Key | 认证密钥 |
| 模型选择 | 使用的大模型 |
| 超时设置 | 请求超时时间 |

### LLM 建议内容

```json
{
  "current_params": {
    "kp": 1.0,
    "ki": 0.1,
    "kd": 0.05
  },
  "system_response": "超调量 15%，建议降低 Kp",
  "recommendations": [
    {
      "parameter": "kp",
      "action": "decrease",
      "reason": "降低比例增益可减少超调"
    },
    {
      "parameter": "kd",
      "action": "increase",
      "reason": "增加微分增益可改善阻尼"
    }
  ]
}
```

## 使用方法

### 启动神经网络调参

1. 进入 NN Tuning 页面
2. 配置网络结构参数
3. 设置学习率等训练参数
4. 点击"开始训练"
5. 观察训练曲线
6. 应用推荐的参数

### 获取 LLM 建议

1. 确保 LLM API 已配置
2. 点击"获取建议"
3. 等待 LLM 分析
4. 查看建议内容
5. 选择接受或忽略建议

## 监控面板

### 训练曲线

| 曲线 | 说明 |
|------|------|
| Loss Curve | 损失函数下降曲线 |
| Accuracy | 参数准确度曲线 |
| Gradient | 梯度变化曲线 |

### 参数轨迹

显示参数在训练过程中的变化轨迹，用于分析收敛行为。

## 注意事项

### 安全提醒

- 神经网络建议仅供参考
- 建议在离线环境中测试后再应用到实际系统
- 某些极端参数可能导致系统不稳定

### 限制条件

- 神经网络调参需要足够的训练数据
- 对于高度非线性系统，建议使用更复杂的网络结构
- 训练时间取决于数据量和网络规模

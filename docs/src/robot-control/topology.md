# Topology - 拓扑视图

> 机器人拓扑结构可视化与配置

## 功能概述

Topology 模块提供机器人拓扑结构的可视化配置和显示功能。

### 主要功能

| 功能 | 说明 |
|------|------|
| 拓扑可视化 | 图形化显示机器人结构 |
| 节点配置 | 配置各节点参数 |
| 连接管理 | 管理节点间连接 |
| 状态监控 | 实时显示节点状态 |

## 拓扑结构

### 典型机器人拓扑

```
                    ┌─────────────┐
                    │   Master    │
                    │  Controller │
                    └──────┬──────┘
                           │
           ┌───────────────┼───────────────┐
           │               │               │
    ┌──────┴──────┐ ┌─────┴─────┐ ┌─────┴─────┐
    │  Joint 1    │ │  Joint 2  │ │  Joint 3  │
    │   Motor     │ │   Motor   │ │   Motor   │
    └─────────────┘ └───────────┘ └───────────┘
```

### 拓扑类型

| 类型 | 说明 |
|------|------|
| 星型拓扑 | 所有节点连接中心节点 |
| 总线拓扑 | 所有节点连接公共总线 |
| 树形拓扑 | 分层结构的树状连接 |
| 环形拓扑 | 节点形成闭环 |

## 配置导入导出

### 导出配置

导出拓扑配置为 JSON 格式：

```json
{
  "topology": {
    "name": "Robot Arm",
    "type": "tree",
    "nodes": [
      {
        "id": "master",
        "type": "master",
        "position": {"x": 400, "y": 50}
      }
    ],
    "connections": [
      {
        "from": "master",
        "to": "joint1",
        "type": "can"
      }
    ]
  }
}
```

### 导入配置

支持导入 JSON 格式配置，快速恢复拓扑结构。

## 帧解析

### 通用解析功能

- **十六进制显示**: 原始数据以 HEX 格式展示
- **ASCII 显示**: 可打印字符以 ASCII 格式展示
- **时间戳**: 每帧数据携带精确时间戳
- **方向标识**: TX (发送) / RX (接收) 区分

### 过滤功能

#### 按协议过滤

```
┌─────────────────────────────────────┐
│ 协议过滤                            │
├─────────────────────────────────────┤
│ ☑ Serial    ☑ TCP    ☐ UDP       │
│ ☑ CAN       ☑ CAN FD ☐ Modbus RTU │
│ ☐ Modbus TCP ☐ USB                 │
└─────────────────────────────────────┘
```

#### 按方向过滤

| 方向 | 说明 |
|------|------|
| TX | 仅显示发送帧 |
| RX | 仅显示接收帧 |
| 双向 | 显示所有帧 |

#### 关键词过滤

支持正则表达式：

```regex
# 过滤包含特定字节
.*81.*23.*

# 过滤特定帧 ID
0x100|0x200

# 过滤特定字符串
GET|POST|HTTP
```

## 事务分析

### 事务定义

事务 = 请求 + 响应（可能包含多个帧）

### 事务配对

| 协议 | 配对方式 |
|------|----------|
| Modbus | 功能码 + 从机地址 + 寄存器地址 |
| CAN | CAN ID + 数据模式 |
| TCP | 端口 + Sequence |
| USB | Transaction ID |

### 事务视图

```
┌─ Transaction List ────────────────────────────────────┐
│ ID   │ Request          │ Response         │ Status │
├──────┼──────────────────┼──────────────────┼────────┤
│ 001  │ 03 00 00 00 08   │ 03 10 01 02...   │ ✅ OK  │
│ 002  │ 03 00 00 00 04   │ -- timeout --    │ ⏱ Wait│
│ 003  │ 01 05 00 00 FF 00 │ 01 05 00 00...  │ ✅ OK  │
└──────┴──────────────────┴──────────────────┴────────┘
```

### 超时设置

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 响应超时 | 等待响应最大时间 | 1000ms |
| 帧间隔超时 | 同一事务帧间最大间隔 | 100ms |

## CAN 协议分析

### CAN 帧格式

```
┌─────────────────────────────────────────────────┐
│ SOF │ CAN ID (11/29bit) │ DLC │ DATA │ CRC │ EOF │
└─────────────────────────────────────────────────┘
```

### CAN 解析项

| 字段 | 说明 |
|------|------|
| SOF | 帧起始 |
| CAN ID | 标识符 (标准/扩展) |
| RTR | 远程传输请求 |
| DLC | 数据长度码 (0-8) |
| DATA | 数据 (0-8 bytes) |
| CRC | 循环冗余校验 |
| ACK | 确认位 |
| EOF | 帧结束 |

### CAN FD 扩展

| 字段 | 说明 |
|------|------|
| ESI | 错误状态指示 |
| FDF | CAN FD 格式指示 |
| BRS | 比特率切换 |
| DLC | 数据长度码 (0-64) |
| DATA | 数据 (0-64 bytes) |

### CANopen 特定解析

- **NMT**: 网络管理报文
- **SDO**: 服务数据对象
- **PDO**: 过程数据对象
- **Heartbeat**: 心跳报文

## Modbus 协议分析

### Modbus RTU 帧

```
┌──────────┬───────┬─────────┬──────┬────────┐
│ 从机地址  │ 功能码 │  数据   │ CRC  │        │
│  1 字节  │ 1 字节 │ N 字节  │ 2 字节│        │
└──────────┴───────┴─────────┴──────┴────────┘
```

### Modbus TCP 帧

```
┌──────────────────────────────────────────┐
│ MBAP Header (7 bytes)                    │
├────────┬──────┬───────┬─────────────────┤
│ TID(2) │ PID  │ Length│ UID             │
├────────┴──────┴───────┴─────────────────┤
│ Function Code │ Data                    │
│     1 字节     │ N 字节                  │
└──────────────────────────────────────────┘
```

### 功能码解析

| 功能码 | 名称 | 解析内容 |
|--------|------|----------|
| 01 | Read Coils | 读取输出状态位 |
| 02 | Read Inputs | 读取输入状态位 |
| 03 | Read Registers | 读取保持寄存器 |
| 04 | Read Input Regs | 读取输入寄存器 |
| 05 | Write Coil | 写入单个线圈 |
| 06 | Write Register | 写入单个寄存器 |
| 15 | Write Coils | 写入多个线圈 |
| 16 | Write Registers | 写入多个寄存器 |

## USB 协议分析

### 支持的 USB 类

| 类 | 描述 | 解析内容 |
|----|------|----------|
| CDC ACM | 虚拟串口 | Line Coding, AT 命令 |
| HID | 人机接口 | Report Descriptor, Report Data |
| MSC | 大容量存储 | BOT Protocol (CBW/CSW) |
| Audio | 音频 | Audio Control/Streaming |
| Video | 视频 | Video Control/Streaming |

### USB 描述符

| 描述符 | 解析内容 |
|--------|----------|
| Device | VID/PID, 设备类 |
| Configuration | 配置值, 功率需求 |
| Interface | 接口号, 备接口数 |
| Endpoint | 端点地址, 类型, 最大包大小 |

```
│ CB Length: Command Block Length         │

```

| 选项 | 说明 |
|------|------|

| 图标 | 类型 | 说明 |

## 串口连接 (Serial)

### 配置参数

| 参数 | 可选值 | 说明 |
|------|--------|------|
| 端口 | COM1-COM256, /dev/ttyUSB0 等 | 串口设备名 |
| 波特率 | 300 - 3,000,000 | 常用: 9600, 115200, 921600 |
| 数据位 | 5, 6, 7, 8 | 常用: 8 |
| 停止位 | 1, 2 | 常用: 1 |
| 校验位 | None, Odd, Even | 常用: None |
| 流控制 | None, Hardware, Software | 常用: None |

### 常用波特率

| 波特率 | 典型应用 |
|--------|----------|
| 9600 | 慢速设备、老旧串口 |
| 115200 | 常用工业设备 |
| 460800 | 高速数据采集 |
| 921600 | 高速串口设备 |
| 1000000 | 高速工业相机 |
| 3000000 | 最大速率 |

### 使用示例

```powershell
# 端口名称 (Windows)
COM1, COM2, COM3 ...

# 端口名称 (Linux)
/dev/ttyUSB0, /dev/ttyUSB1, /dev/ttyS0 ...
```

## TCP 连接

### 客户端模式

| 参数 | 说明 |
|------|------|
| 目标 IP | 服务器 IP 地址 |
| 目标端口 | 服务器端口号 |
| 连接超时 | 超时时间 (ms) |

### 服务端模式

| 参数 | 说明 |
|------|------|
| 监听端口 | 本机监听端口 |
| 最大连接数 | 允许的客户端数量 |

### 示例配置

```powershell
# TCP 客户端
IP: 192.168.1.100
Port: 8080
Timeout: 5000ms

# TCP 服务端
Listen Port: 8080
Max Connections: 5
```

## UDP 连接

### 配置参数

| 参数 | 说明 |
|------|------|
| 目标 IP | 目标主机地址 (单播) |
| 目标端口 | 目标端口号 |
| 本机端口 | 本机监听端口 (可选) |
| 广播模式 | 启用 UDP 广播 |

### 广播配置

启用广播后，数据将发送到局域网广播地址：

```powershell
# 广播 IP (根据网段)
Broadcast: 192.168.1.255
Port: 5000
```

## CAN 2.0 连接

### 配置参数

| 参数 | 可选值 | 说明 |
|------|--------|------|
| 波特率 | 125k, 250k, 500k, 1M | CAN 总线速率 |
| 采样点 | 75%, 87.5% | 采样时刻位置 |
| SJW | 1-4 | 同步跳转宽度 |
| 协议模式 | Standard / Extended | 标准帧/扩展帧 |

### 常用波特率配置

| 波特率 | Tq 数量 | 典型采样点 |
|--------|---------|-----------|
| 125k | 16 | 87.5% |
| 250k | 16 | 87.5% |
| 500k | 16 | 87.5% |
| 1M | 16 | 75% |

### 帧类型

- **标准帧**: 11-bit CAN ID
- **扩展帧**: 29-bit CAN ID

## CAN FD 连接

### 与 CAN 2.0 区别

| 特性 | CAN 2.0 | CAN FD |
|------|---------|--------|
| 最大速率 | 1Mbps | 8Mbps |
| 最大 Payload | 8 bytes | 64 bytes |
| 协议层 | 相同 | 相同 |
| 控制器 | 普通 CAN | 需要 CAN FD |

### 配置参数

| 参数 | 可选值 | 说明 |
|------|--------|------|
| 仲裁波特率 | 125k-1M | 总线仲裁阶段速率 |
| 数据波特率 | 1M-8M | 数据阶段速率 |
| 采样点 | 75%, 87.5% | 数据阶段采样点 |

## Modbus RTU

### 配置参数

| 参数 | 说明 |
|------|------|
| 从机地址 | 1-247 |
| 波特率 | 9600, 19200, 38400, 57600 |
| 数据位 | 8 |
| 停止位 | 1 或 2 |
| 校验位 | None, Odd, Even |

### 寄存器类型

| 类型 | 功能码 | 说明 |
|------|--------|------|
| 线圈 (Coil) | 01, 05, 15 | 读写单个/多个位 |
| 离散输入 | 02 | 只读位 |
| 保持寄存器 | 03, 06, 16 | 读写单个/多个 16-bit 字 |
| 输入寄存器 | 04 | 只读 16-bit 字 |

### 功能码

| 功能码 | 名称 | 操作 |
|--------|------|------|
| 01 | Read Coils | 读多个线圈 |
| 02 | Read Discrete Inputs | 读离散输入 |
| 03 | Read Holding Registers | 读保持寄存器 |
| 04 | Read Input Registers | 读输入寄存器 |
| 05 | Write Single Coil | 写单个线圈 |
| 06 | Write Single Register | 写单个寄存器 |
| 15 | Write Multiple Coils | 写多个线圈 |
| 16 | Write Multiple Registers | 写多个寄存器 |

## Modbus TCP

### 配置参数

| 参数 | 说明 |
|------|------|
| 从机地址 | IP 地址 |
| 端口号 | 默认 502 |
| 从机 ID | Modbus 从机标识 (0-255) |

### 帧结构

```
┌─────────┬──────┬──────┬─────────┬───────┬──────────┐
│  MBAP  │ UID  │ FUNC │  DATA  │ CRC   │          │
│  7 字节 │ 1 字节│ 1 字节│ N 字节  │ 2 字节 │          │
└─────────┴──────┴──────┴─────────┴───────┴──────────┘
```

- **MBAP**: Modbus Application Header (事务ID、协议ID、长度、单元ID)
- **UID**: 单元标识符
- **Func**: 功能码

## 常见问题

### 串口无法打开

1. 检查端口名称是否正确
2. 确认端口未被其他程序占用
3. 检查设备是否已连接
4. 确认驱动程序已安装

### TCP 连接超时

1. 检查目标 IP 和端口
2. 确认目标服务器已启动
3. 检查防火墙设置
4. 验证网络连通性

### CAN 总线错误

1. 检查波特率设置是否匹配
2. 确认终端电阻 (120Ω) 已连接
3. 检查 CAN_H / CAN_L 线序
4. 验证总线负载是否过高

# Packet Builder - 报文构建器

> 模板式报文构建与二进制解析工具

## 功能概述

Packet Builder 提供可视化的报文构建和解析功能，支持 14 种字段类型和 6 种校验算法。

### 字段类型

| 类型 | 大小 | 说明 |
|------|------|------|
| u8 | 1 byte | 无符号 8 位整数 |
| i8 | 1 byte | 有符号 8 位整数 |
| u16 | 2 bytes | 无符号 16 位整数 |
| i16 | 2 bytes | 有符号 16 位整数 |
| u32 | 4 bytes | 无符号 32 位整数 |
| i32 | 4 bytes | 有符号 32 位整数 |
| u64 | 8 bytes | 无符号 64 位整数 |
| i64 | 8 bytes | 有符号 64 位整数 |
| f32 | 4 bytes | 单精度浮点数 |
| f64 | 8 bytes | 双精度浮点数 |
| bool | 1 byte | 布尔值 |
| string | N bytes | 字符串 |
| hex | N bytes | 十六进制数据 |
| timestamp | 8 bytes | 时间戳 |

### 校验算法

| 算法 | 输出长度 | 说明 |
|------|----------|------|
| XOR | 1 byte | 异或校验 |
| CRC8 | 1 byte | CRC-8 |
| CRC16 | 2 bytes | CRC-16 (Modbus) |
| CRC32 | 4 bytes | CRC-32 |
| Modbus CRC | 2 bytes | Modbus CRC |
| Checksum | 2 bytes | 简单求和校验 |

## 模板管理

### 创建模板

1. 点击"新建模板"
2. 输入模板名称
3. 添加字段
4. 配置字段属性
5. 保存模板

### 模板属性

| 属性 | 说明 |
|------|------|
| 名称 | 模板唯一标识 |
| 描述 | 模板说明 |
| 字节序 | 大端/小端 |
| 校验位置 | 校验字段在帧中的位置 |

## 字段配置

### 字段属性

| 属性 | 可选值 | 说明 |
|------|--------|------|
| 名称 | 用户定义 | 字段显示名称 |
| 类型 | 14 种类型 | 数据类型 |
| 长度 | 取决于类型 | 字节数 |
| 字节序 | LE/BE | Little-endian / Big-endian |
| 初始值 | 用户定义 | 默认值 |
| 约束 | min/max | 取值范围 |

### 字节序配置

| 字节序 | 说明 | 示例 (0x1234) |
|--------|------|----------------|
| Big-endian | 高字节在前 | 12 34 |
| Little-endian | 低字节在前 | 34 12 |
| Native | 系统字节序 | 取决于平台 |

## 报文构建

### 构建流程

```
1. 选择模板
2. 填充字段值
3. 计算校验和
4. 生成最终报文
```

### 构建示例

```
模板: MotorControl
字段:
  - Header: 0xAA (u8, 初始值)
  - ID: 0x01 (u8)
  - Speed: 1000 (i16, big-endian)
  - Accel: 50 (u16, big-endian)
  - CRC: 自动计算 (CRC16)

最终报文:
AA 01 03 E8 00 32 XX XX
```

## 报文解析

### 解析流程

```
1. 输入原始报文
2. 选择解析模板
3. 自动解析字段
4. 显示解析结果
```

### 解析结果示例

```
原始报文: AA 01 03 E8 00 32 12 34

使用模板 MotorControl 解析:
┌──────────┬────────┬─────────┐
│ 字段     │ 值     │ 说明     │
├──────────┼────────┼─────────┤
│ Header   │ 0xAA   │ 帧头     │
│ ID       │ 0x01   │ 电机 ID  │
│ Speed    │ 1000   │ 速度 rpm │
│ Accel    │ 50     │ 加速度   │
│ CRC      │ 0x1234 │ 校验     │
└──────────┴────────┴─────────┘
```

## 报文导入导出

### 模板导出

导出模板为 JSON 格式：

```json
{
  "name": "MotorControl",
  "description": "电机控制报文",
  "endian": "big",
  "fields": [
    {"name": "Header", "type": "u8", "value": "0xAA"},
    {"name": "ID", "type": "u8", "value": "0x01"},
    {"name": "Speed", "type": "i16", "value": "1000"},
    {"name": "Accel", "type": "u16", "value": "50"}
  ],
  "checksum": {
    "type": "CRC16",
    "position": "last"
  }
}
```

### 模板导入

支持导入 JSON 格式模板，快速复用已有配置。

## 使用场景

### 调试与测试

快速构造测试报文，验证设备响应。

### 协议逆向

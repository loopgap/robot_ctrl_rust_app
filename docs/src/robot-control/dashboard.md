# Dashboard - 仪表盘

> 系统状态总览与快捷操作入口

## 功能概述

Dashboard 是应用的主页面，提供系统状态总览和快捷操作入口。

### 主要功能

| 功能 | 说明 |
|------|------|
| 连接状态卡片 | 实时显示 Serial/TCP/UDP/CAN 连接状态 |
| 系统自检 | 启动时自动检查配置和环境 |
| 运行时指标 | 显示连接数、LLM 请求、MCP 连接数 |
| 更新检查 | 检测新版本并提示 |
| 快捷动作 | 连接、启动/停止控制、急停 |

## 连接状态卡片

### 显示的连接类型

| 连接 | 状态指示 |
|------|----------|
| Serial | 端口名称、波特率、连接状态 |
| TCP | 客户端/服务端、IP:Port、状态 |
| UDP | 单播/广播模式、状态 |
| CAN | 运行/停止状态 |

### 状态颜色

| 状态 | 颜色 | 说明 |
|------|------|------|
| 已连接 | 🟢 绿色 | 正常运行 |
| 连接中 | 🟡 黄色 | 正在建立连接 |
| 已断开 | ⚪ 灰色 | 未连接 |
| 错误 | 🔴 红色 | 连接异常 |

## System Check - 系统自检

### 检查项目

| 检查项 | 说明 |
|--------|------|
| 偏好路径 | 配置文件目录是否存在 |
| 日志路径 | 日志文件目录是否可写 |
| MCP 端口 | 端口是否被占用 |
| LLM URL | API 端点可达性 |
| 串口可用性 | 是否有可用串口设备 |

### 检查结果显示

```
Passed: 5/6  |  Version: 0.1.1
```

- **Passed**: 通过的检查项数量
- **Total**: 总检查项数量
- **Version**: 当前应用版本

## Runtime Metrics - 运行时指标

### 监控指标

| 指标 | 说明 |
|------|------|
| 连接数 | 当前活动的连接数量 |
| LLM 请求 | LLM API 调用次数 |
| MCP 连接 | MCP Server 连接数 |
| 控制状态 | 运行/停止状态 |

### 实时更新

指标每秒钟更新一次，显示当前会话的累计值。

## Update Check - 更新检查

### 版本策略

- **预发布版本**: `0.1.x` 版本视为功能/变更发布
- **补丁版本**: `0.x.patch` 版本视为修复
- **检查间隔**: 可配置检查频率
- **Manifest URL**: 可配置更新清单地址

### 更新状态显示

```
Update: Available | Last check: 2024-01-15 10:30:00
```

- **Update Available**: 有可用更新
- **Up to date**: 已是最新版本
- **Check failed**: 检查失败

## 快捷动作

### 控制按钮

| 按钮 | 功能 |
|------|------|
| 连接 | 打开连接管理页面 |
| 启动/停止 | 切换控制算法运行状态 |
| 急停 | 立即停止所有控制输出 |
| 刷新端口 | 重新扫描可用串口 |

### 快捷键

| 操作 | 快捷键 |
|------|--------|
| 启动控制 | Ctrl+Shift+S |
| 停止控制 | Ctrl+Shift+X |
| 急停 | Ctrl+Shift+E |
| 刷新端口 | F5 |

## 状态栏信息

### 底部状态栏

```
Serial: Connected (COM3, 115200) | TCP: 2 clients | Language: 中文
```

显示内容：
- Serial 连接状态
- TCP 客户端数量
- 当前语言设置
- 性能指标（可选）

## 经典 PID

### 算法原理

位置式 PID 输出：

```
u(t) = Kp × e(t) + Ki × ∫e(t)dt + Kd × de(t)/dt
```

其中：
- `Kp`: 比例增益
- `Ki`: 积分增益
- `Kd`: 微分增益
- `e(t)`: 设定值与实际值之差

### 参数作用

| 参数 | 作用 | 增大效果 |
|------|------|----------|
| Kp | 响应速度 | 更快、更振荡 |
| Ki | 稳态误差消除 | 更快消除静差、可能超调 |
| Kd | 阻尼效果 | 减少超调、增加稳定性 |

### 调参口诀

```
参数整定找最佳，从小到大顺序查
先是比例后积分，最后才把微分加
曲线振荡很频繁，比例度盘要放大
曲线漂浮绕大弯，比例度盘往小扳
曲线与横轴相交点，积分时间往下降
曲线波动周期长，积分时间再加长
理想曲线两个峰，前高后低四比一
一看二调多分析，调节质量不会低
```

## 增量 PID

### 算法原理

增量式 PID 输出控制增量 `Δu(t)`：

```
Δu(t) = Kp × (e(t) - e(t-1)) + Ki × e(t) + Kd × (e(t) - 2e(t-1) + e(t-2))
```

### 优势

- **无积分饱和**: 累加器只输出增量
- ** Bumpless 切换**: 切换时无冲击
- **计算量小**: 适合嵌入式

### 适用场景

- 执行器有输出限制
- 需要无扰切换
- 资源受限的嵌入式系统

## Bang-Bang 控制

### 算法原理

最简单的开关控制：

```
u(t) = Umax, 如果 e(t) > 阈值
u(t) = 0,   如果 e(t) ≤ 阈值
```

### 配置参数

| 参数 | 说明 |
|------|------|
| 阈值 | 开关切换的误差阈值 |
| 滞环宽度 | 防止频繁切换 |
| 最大输出 | 开启时的输出值 |

### 适用场景

- 温度控制（加热器）
- 液位控制
- 简单逻辑控制

## 模糊 PID

### 算法原理

模糊 PID 使用模糊逻辑自动调整 PID 参数：

1. **模糊化**: 将误差和误差变化率转换为模糊语言变量
2. **模糊推理**: 根据规则库推理输出
3. **去模糊化**: 将模糊输出转换为精确参数调整量

### 参数

| 参数 | 说明 |
|------|------|
| Kp 模糊规则 | Kp 调整规则 |
| Ki 模糊规则 | Ki 调整规则 |
| Kd 模糊规则 | Kd 调整规则 |
| 隶属函数 | 输入输出隶属度定义 |

### 适用场景

- 非线性系统
- 时变系统
- 难以建立精确数学模型

## 串级 PID

### 算法原理

双闭环控制：
- **外环 (位置环)**: 控制最终目标
- **内环 (速度环)**: 控制中间变量

```
位置环输出 → 速度环设定值 → 速度环输出 → 执行器
```

### 参数整定

| 环 | 参数 | 整定顺序 |
|----|------|----------|
| 内环 | Kp, Ki, Kd | 先整定内环 |
| 外环 | Kp, Ki, Kd | 后整定外环 |

### 适用场景

- 电机速度/位置控制
- 过程控制（温度、压力）
- 需要更快响应的系统

## Smith 预估控制

### 问题背景

对于有纯滞后的系统：

```
G(s) = Gp(s) × e^(-τs)
```

普通 PID 难以处理，Smith 预估器通过内部模型补偿滞后。

### 算法原理

```
1. 建立过程数学模型 Gp(s)
2. 构建预估器 Gp(s) × (1 - e^(-τs))
3. 将预估输出与实际输出比较
4. 用误差修正控制量
```

### 参数

| 参数 | 说明 |
|------|------|
| 过程增益 | Kp |
| 时间常数 | T |
| 纯滞后时间 | τ |

### 适用场景

- 管道流量控制
- 长距离传输系统
- 网络控制系统

## ADRC (自抗扰控制)

### 算法组成

```
┌─────────────────────────────────────────────┐
│                 ADRC 结构                    │
├─────────────────────────────────────────────┤
│  跟踪微分器 (TD)     - 安排过渡过程          │
│  扩张状态观测器 (ESO) - 估计扰动              │
│  非线性误差反馈 (NLSEF) - 生成控制量          │
└─────────────────────────────────────────────┘
```

### 核心思想

将**未知扰动**和**已知模型**一起视为"总扰动"，通过 ESO 实时估计并补偿。

### 参数

| 参数 | 说明 | 调整原则 |
|------|------|----------|
| 跟踪速度 | TD 跟踪速度 | 越快响应越快，但可能振荡 |
| 观测器带宽 | ESO 带宽 | 影响扰动估计速度 |
| 非线性增益 | NLSEF 增益 | 影响控制强度 |

### 适用场景

- 高性能伺服控制
- 鲁棒性要求高的系统
- 存在未知扰动

## LADRC (线性自抗扰控制)

### 与 ADRC 区别

将非线性 ADRC 线性化，参数更少、整定更简单。

### 参数

| 参数 | 说明 |
|------|------|
| 控制器带宽 | ωc |
| 观测器带宽 | ωo |
| 扰动估计增益 | b0 |

### 整定公式

```
Kp = ωc²
Ki = ωc³
Kd = 2ζωc
```

其中 ζ 通常取 0.7-1.0。

## LQR (线性二次调节器)

### 优化目标

最小化二次性能指标：

```
J = ∫(x'Qx + u'Ru)dt
```

其中：
- `Q`: 状态权重矩阵
- `R`: 控制权重矩阵
- `x`: 状态向量
- `u`: 控制向量

### 参数

| 参数 | 说明 |
|------|------|
| Q 矩阵 | 状态权重 |
| R 矩阵 | 控制权重 |
| 终端权重 | 末状态权重 |

### 适用场景

- 航天器控制
- 机器人关节控制
- 最优控制问题

## MPC (模型预测控制)

### 核心思想

1. **预测模型**: 预测未来 N 步的系统行为
2. **优化**: 求解有限时域内的最优控制序列
3. **滚动优化**: 只执行第一步，然后重复

### 参数

| 参数 | 说明 | 典型值 |
|------|------|--------|
| 预测时域 | Np | 10-50 |
| 控制时域 | Nc | 1-10 |
| 权重矩阵 | Q, R | 调试确定 |

### 适用场景

- 多约束系统
- 多变量系统
- 需要处理约束的控制

## 算法选择指南

### 场景对照

| 场景 | 推荐算法 |
|------|----------|
| 简单温度控制 | Bang-Bang / PID |
| 电机位置控制 | 串级 PID / LQR |
| 高精度伺服 | ADRC / MPC |
| 时滞系统 | Smith / MPC |
| 非线性系统 | 模糊 PID / ADRC |
| 资源受限 | 增量 PID / LADRC |

### 选型决策树

```
开始
  │
  ├─ 简单开关控制？ → Bang-Bang
  │
  ├─ 需要高精度？ → ADRC / MPC
  │
  ├─ 有纯滞后？ → Smith / MPC
  │
  └─ 通用场景 → PID / 增量 PID
```

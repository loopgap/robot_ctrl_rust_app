# PID Control - PID 控制页面

> 控制算法选择、参数配置与实时监控

## 功能概述

PID Control 页面提供控制算法的选择、参数配置和实时监控功能。

│  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐         │

## 控制按钮

### 启动/停止控制

| 按钮 | 快捷键 | 功能 |
|------|--------|------|
| 启动控制 | Ctrl+Shift+S | 开始控制算法 |
| 停止控制 | Ctrl+Shift+X | 停止控制算法 |
| 急停 | Ctrl+Shift+E | 立即停止所有输出 |

### 状态指示

| 状态 | 颜色 | 说明 |
|------|------|------|
| 运行中 | 绿色 | 控制算法正在运行 |
| 已停止 | 灰色 | 控制已停止 |
| 急停 | 红色闪烁 | 急停触发 |

## 算法选择

### 支持的算法

| 算法 | 说明 |
|------|------|
| 经典 PID | 标准位置式 PID |
| 增量 PID | 增量式 PID |
| Bang-Bang | 开关控制 |
| 模糊 PID | 自适应模糊 PID |
| 串级 PID | 双闭环 PID |
| Smith 预估 | 时滞补偿 |
| ADRC | 自抗扰控制 |
| LADRC | 线性自抗扰 |
| LQR | 线性二次调节 |
| MPC | 模型预测控制 |

### 算法切换

点击算法按钮即可切换当前控制算法。切换时参数保持不变。

## 参数配置

### 通用参数

| 参数 | 说明 | 典型范围 |
|------|------|----------|
| Kp | 比例增益 | 0.1 - 100 |
| Ki | 积分增益 | 0.01 - 10 |
| Kd | 微分增益 | 0.01 - 10 |

### 高级参数

| 参数 | 说明 |
|------|------|
| 设定值 (SP) | 目标值 |
| 采样周期 | 控制周期 |
| 输出限幅 | 输出上下限 |
| 微分滤波 | 微分项滤波系数 |

### 参数调整

| 操作 | 方式 |
|------|------|
| 直接输入 | 在输入框中输入数值 |
| 滑块调节 | 拖动滑块微调 |
| 旋钮调节 | 鼠标滚轮调整 |

## 监控显示

### 实时数据

| 显示项 | 说明 |
|------|------|
| 设定值 (SP) | 当前目标值 |
| 当前值 (PV) | 过程变量 |
| 输出 (OP) | 控制输出值 |
| 误差 (Error) | SP - PV |
| 积分项 | 积分累积值 |

### 实时曲线

| 曲线 | 颜色 | 说明 |
|------|------|------|
| 设定值 | 绿色 | 目标值直线 |
| 当前值 | 蓝色 | 实际过程值 |
| 输出 | 红色 | 控制输出 |

## 调参模式

### 手动调参

手动调整 Kp, Ki, Kd，观察系统响应。

### 自动调参

点击"自动调参"启动自动参数整定功能。

### NN 调参

调用神经网络模块获取参数建议。

### LLM 调参

获取 LLM 的参数调整建议。

## 报警与保护

### 报警限

| 类型 | 说明 |
|------|------|
| 上限报警 | PV 超过上限 |
| 下限报警 | PV 低于下限 |
| 偏差报警 | PV 与 SP 偏差过大 |

### 安全保护

| 功能 | 说明 |
|------|------|
| 积分限幅 | 防止积分饱和 |
| 输出限幅 | 限制最大输出 |
| 微分限幅 | 防止微分突变 |
| 急停 | 紧急停止 |
